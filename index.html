结构标准语言

  1. 可扩展标记语言 —-XML

    • Schema

    • XMLHttpRequest对象

      • 在后台与服务器交换数据

    • XML解析器

    • XML DOM

      • document.getElementById(“to”).innerHTML=
        xmlDoc.getElementsByTagName(“to”)[0].childNodes[0].nodeValue;

  1. 可扩展超文本编辑语言 —-XHTML

    • 特点：
      • XHTML 元素必须被正确地嵌套。
      • XHTML 元素必须被关闭。
      • 标签名必须用小写字母。
      • XHTML 文档必须拥有根元素

Hexo博客安装

2020-08-03

安装nodejs

  • node -v

安装cnpm

  • npm install -g cnpm –registry=https://registry.npm.taobao.org

安装hexo组件，启动博客

  1. cnpm install -g hexo-cli
  2. mkdir blog
  3. cd blog
  4. hexo init
  5. hexo s //启动
  6. hexo n “xxx” //新建博文
  7. hexo clean //清理一下
  8. hexo g //重新生成一下
  9. hexo s //再启动看效果

配到github

  1. 注册github账号
  2. new repository
    名字必须是 github名.github.io
  3. create
  4. cnpm install –save hexo-deployer-git
  5. 编辑_config.yml
    最后一行，deploy:
type:git
repo: https://github.com/xxx/xxx.github.io.git
branch: master
  6. 保存
  7. hexo d
  8. 输入账号，密码
  9. 刷新，博客的内容就出来了
  10. 访问https://xxx.github.io 就能看见你的博客了

自定义主题

  1. 在网上找到要的hexo主题的github地址
  2. git clone xxxxx themes/xxx
  3. 修改_config.yml
  4. 找到theme那一行
    theme: xxx
  5. hexo clean
  6. hexo g
  7. hexo s
    主题就上去了
  8. hexo d
    等一会儿，访问github.io就行了

sql server证书加密解密

2020-08-01

加密前

  • drop 了证书和对称密钥

步骤

  • use dbname;
  • –创建master key
  • CREATE MASTER KEY ENCRYPTION BY PASSWORD =’password’;
  • use dbname;
  • –创建证书
  • CREATE CERTIFICATE 证书名字
  • with SUBJECT = ‘Certificate’
  • GO
  • use dbname;
  • –由证书加密对称密钥
  • CREATE SYMMETRIC KEY 密钥名字
  • WITH ALGORITHM = AES_256
  • ENCRYPTION BY CERTIFICATE 证书名字;
  • GO
  • use dbname;
  • –打开对称密钥
  • OPEN SYMMETRIC KEY 密钥名字 DECRYPTION BY CERTIFICATE 证书名字;

  • https://www.cnblogs.com/wolfocme110/p/4016035.html

computed与methods

computed与methods效果一样
但，computed基于他的缓存依赖，依赖发生改变时才会重新取值
而，methods在重新渲染时，总会重新调用执行
Computed将function与里面的值绑定，只有在值变化时才会触发content,
methods每次进入页面都会执行该方法
