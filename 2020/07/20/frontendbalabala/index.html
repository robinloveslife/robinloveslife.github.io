开发工具

  • sublime

    • 快捷键
      • html:xt|5+tab 快速生成html骨架
      • 标签名+tab 快速生成标签对
      • p*数量+tab 快速生成一定数量的同种标签
      • 按住滚轮拖动 选中多行进行同时编辑
      • shift+ctrl+↑ 上移当前行
      • shift+ctrl+↓ 下移当前行
      • ctrl+shift+S 另存为
      • ctrl+/ 注释
  • vs code
    • 快捷键
      • ！+tab / html:xt|5+tab 快速生成html骨架
      • 标签名+tab 快速生成标签对
      • p*数量+tab 快速生成一定数量的同种标签
      • alt+shift+鼠标左键 选中多行进行同时编辑
      • alt+↑ 上移当前行
      • alt+↓ 下移当前行
      • ctrl+shift+S 另存为
      • ctrl+/ 注释

HTML

DTD

(DocType Definition)文档类型定义

字符集

国际网站 用utf-8

国内，有大量中文 用GBK/gb2312

属性

  • 必需属性 如 img里的src alt
  • 可选属性 如 img里的width height
  • 标准属性 如id class style lang
  • 事件属性 如 onclick onmouseover

路径

  • 相对路径：目标文件相对引用文件所在位置 / ./ ../
  • 绝对路径： 目标文件所在完全路径 d:/xxx/xxx

超链接

  • href：地址
  • title：提示信息
  • rel：规定当前文档与被链接文档之间关系
  • download：规定被下载的超链接的目标
  • media：规定被链接文档为何种媒介优化
  • type：规定被链接文档的MIME类型
  • target：
    • _self：在当前页面打开新页面
    • _blank：在新的页面打开
    • _parent：页面的父窗口打开
    • _top：将页面在整个浏览器窗口打开
    • _framename 在指定的框架中打开
