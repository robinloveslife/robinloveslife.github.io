Hexo博客安装

2020-08-03

安装nodejs

  • node -v

安装cnpm

  • npm install -g cnpm –registry=https://registry.npm.taobao.org

安装hexo组件，启动博客

  1. cnpm install -g hexo-cli
  2. mkdir blog
  3. cd blog
  4. hexo init
  5. hexo s //启动
  6. hexo n “xxx” //新建博文
  7. hexo clean //清理一下
  8. hexo g //重新生成一下
  9. hexo s //再启动看效果

配到github

  1. 注册github账号
  2. new repository
    名字必须是 github名.github.io
  3. create
  4. cnpm install –save hexo-deployer-git
  5. 编辑_config.yml
    最后一行，deploy:
type:git
repo: https://github.com/xxx/xxx.github.io.git
branch: master
  6. 保存
  7. hexo d
  8. 输入账号，密码
  9. 刷新，博客的内容就出来了
  10. 访问https://xxx.github.io 就能看见你的博客了

自定义主题

  1. 在网上找到要的hexo主题的github地址
  2. git clone xxxxx themes/xxx
  3. 修改_config.yml
  4. 找到theme那一行
    theme: xxx
  5. hexo clean
  6. hexo g
  7. hexo s
    主题就上去了
  8. hexo d
    等一会儿，访问github.io就行了
